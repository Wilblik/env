* WSL Checkpoints & Recovery (for development)

#+begin_caption
Create a snapshot:
#+end_caption
#+begin_src powershell
wsl --terminate archlinux
wsl --export archlinux ".\archlinux-vanilla-checkpoint.tar"
#+end_src

#+begin_caption
Restore from snapshot:
#+end_caption
#+begin_src powershell
wsl --unregister archlinux
wsl --import archlinux "C:\WSL\archlinux" ".\archlinux-vanilla-checkpoint.tar"
#+end_src
